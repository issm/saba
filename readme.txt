.htaccessの設定
----------------------------------------------------------------
RewriteBase

http://www.example.com/hoge/で動作させる場合:

    RewriteBase /hoge/


----------------------------------------------------------------
etc/conf.ymlの設定
----------------------------------------------------------------

■ロケーション設定

http://www.example.com/hoge/で動作させる場合:

    LOCATION:
      PROTOCOL:  http
      DOMAIN:    example.com
      SUBDOMAIN: www.   # ピリオドを忘れずに！
      PATH:      /hoge  # 最後のスラッシュは不要


■データベース設定

    DB:
      TYPE:   <データベースタイプ>  # 現状はmysqlのみ
      NAME:   <データベース名>
      HOST:   <データベースサーバ>  # 原則localhost
      USER:   <ユーザ名>
      PASSWD: <パスワード>
      TABLE_PREFIX: <テーブル接頭辞>
