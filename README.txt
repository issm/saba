設置
----------------------------------------------------------------
  Base URL:     http://www.example.com/
  Target URL:   http://www.example.com/hoge/
  DocumentRoot: /var/www/htdocs/ 

上記のような環境の場合:

% cd /var/www/htdocs
% mkdir hoge
% cd hoge
% git clone git://github.com/issm/saba.git



----------------------------------------------------------------
セットアップスクリプトの実行
----------------------------------------------------------------

■コマンドラインから

% ./saba/setup.cgi
or
% perl ./saba/setup.cgi


■ブラウザから

http://www.example.com/hoge/下にインストールした場合:

http://www.example.com/hoge/saba/setup.cgi



----------------------------------------------------------------
.sabaの設定
----------------------------------------------------------------

■ロケーション設定

http://www.example.com/hoge/で動作させる場合:

    LOCATION:
      PROTOCOL:  http
      DOMAIN:    example.com
      SUBDOMAIN: www.   # ピリオドを忘れずに！
      PATH:      /hoge  # 最後のスラッシュは不要


■データベース設定

    DB:
      TYPE:   <データベースタイプ>  # 現状はmysqlのみ
      NAME:   <データベース名>
      HOST:   <データベースサーバ>  # 基本localhost
      USER:   <ユーザ名>
      PASSWD: <パスワード>
      TABLE_PREFIX: <テーブル接頭辞>
